# Nodejs
## Iniciar proyecto node
```
npm init
```
## Ejecutar js (con o sin extensi√≥n) con node
```
node index
node src/index
```

## Ejecutar ts con node
```
npm i --save ts-node
ts-node <<nombre_archivo>>
```

## Instalar definiciones de node
```
npm i --save-dev @types/node
```

## Instalar express
````
npm i --save express
npm i --save-dev @types/express
```
